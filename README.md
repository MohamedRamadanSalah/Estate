# عقارك - منصة عقارات مصر

منصة عقارية عربية متكاملة مصممة خصيصاً للسوق المصري. تدعم البحث عن العقارات في القاهرة، الجيزة، الإسكندرية ومطروح.

## المتطلبات

- Node.js 18+
- PostgreSQL
- حساب Cloudinary (لرفع الصور)

## التثبيت

```bash
# تثبيت الحزم
npm install

# نسخ ملف البيئة
cp .env.example .env

# تعديل .env بالقيم الصحيحة
# DATABASE_URL - رابط قاعدة بيانات PostgreSQL
# JWT_SECRET - مفتاح سري للتوقيع
# Cloudinary - بيانات الحساب
# NEXT_PUBLIC_WHATSAPP_NUMBER - رقم الواتساب (مثال: 201234567890)
```

## إعداد قاعدة البيانات

```bash
# إنشاء الجداول
npx prisma db push

# أو باستخدام migrations
npx prisma migrate dev

# استيراد المحافظات والمدن والمسؤول الافتراضي
npm run db:seed
```

بيانات الدخول الافتراضية (يمكن تغييرها في .env):
- البريد: admin@example.com
- كلمة المرور: admin123

## التشغيل

```bash
# وضع التطوير
npm run dev

# البناء للإنتاج
npm run build

# تشغيل الإنتاج
npm start
```

## هيكل المشروع

```
src/
├── app/
│   ├── page.tsx              # الصفحة الرئيسية
│   ├── properties/           # عرض الوحدات
│   ├── properties/[unitCode] # تفاصيل الوحدة
│   ├── search/               # نتائج البحث

│   │   ├── login/            # تسجيل دخول المسؤول
│   │   └── (auth)/           # لوحة التحكم
│   └── api/                  # مسارات API
├── components/
├── lib/
└── types/
```

## الميزات

- **الصفحة الرئيسية**: Hero مع بحث، إحصائيات، تصفح بالمحافظة، أحدث الوحدات
- **عرض الوحدات**: فلترة متقدمة، ترتيب، تنقل بين الصفحات
- **تفاصيل الوحدة**: معرض صور، خريطة، حاسبة تقسيط، زر واتساب
- **لوحة التحكم**: إدارة الوحدات، المدن، المطورين
- **كود الوحدة**: يُولَّد تلقائياً بصيغة CAI-APT-TAG-2502-0047
